前端界面优化的渐进方案

2026-05-18前端CSSUX

患者 Agent 的 tester.html 页面做了一次纯视觉层面的渐进优化，不改变任何业务功能、API 调用和数据结构。

优化清单

视觉层次

  • 主面板暖白背景（rgba 0.84），次要面板降为 0.6 透明度，建立视觉主次
  • 用户气泡：纯色 → 品牌绿渐变
  • 助手气泡：纯灰 → 微暖色
  • 空状态提示美化：字号、字距、行高调整

间距体系统一

统一为 8 的倍数体系：8 / 12 / 16 / 24 / 32。原来 gap 混用 10/12/14/16/18 → 统一 12 或 16。margin-top 混用 6/8/10/12/14/16/18 → 统一 8 或 16。

动效与交互

  • 发送按钮：disabled + 脉冲动画 + 文字"发送中…"，完成后恢复
  • 面板展开/收起：CSS grid-template-rows 过渡动画
  • 空状态到有内容：淡出过渡
  • 所有动画在 prefers-reduced-motion 下禁用

约束

不改变任何 HTML 语义 class 名，不改变 JS 状态结构、API 调用、事件绑定逻辑，不引入外部依赖或新文件，所有改动在 tester.html 一个文件内完成。
