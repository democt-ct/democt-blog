## 项目定位

对话驱动的智能旅行规划系统，用户通过自然语言输入需求即可生成完整的结构化行程，并自动映射到高德地图。

## 核心能力

- **聊天生成行程**：用户输入 → LLM 生成结构化 iternery
- **高德 POI 落图**：自动解析地点并获取坐标，按天展示在地图上
- **路径规划**：每天从第 1 个行程点开始做路线规划，2D/3D 模式切换
- **泛地点自动补点**：对「咖啡馆/商圈/美食街」等模糊需求自动推荐候选 POI
- **行程编辑**：备选点直接加入指定日期，支持同日内拖拽排序
- **地点搜索**：手动搜索与当前行程共存
- **会话管理**：多会话、历史恢复、行程快照保存
- **智能校验**：跨天重复地址检测与修复

## 技术栈

| 组件 | 技术 |
|------|------|
| 后端框架 | FastAPI |
| 地图服务 | 高德地图 SDK（浏览器端 + 服务端 API） |
| 地点解析 | 自定义 POI 解析 pipeline |
| 会话存储 | SQLite |
| 前端 | HTML/CSS/JS 单页应用 |
| LLM | 后端大模型驱动行程生成 |
